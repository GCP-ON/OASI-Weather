# OASI Weather

OASI Weather is a simple weather application designed to fetch and display current weather data for specified locations.

## Features

- Fetches real-time weather data
- Displays temperature, humidity, and conditions
- Supports multiple locations

## Installation

```bash
git clone https://github.com/yourusername/oasi-weather.git
cd oasi-weather
npm install
```

## Usage

```bash
npm start
```

Configure your API keys and location preferences in the `.env` file.

## Configuration

Create a `.env` file with the following:

```env
WEATHER_API_KEY=your_api_key
DEFAULT_LOCATION=your_city
```
